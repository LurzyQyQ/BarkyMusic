# barkymusic

### 环境要求 需要 NodeJS 8.12+ 环境

### 下载

```
$ git https://github.com/LurzyQyQ/barkymusic_player_vue.git

$ npm install
```

### 运行
```
$ npm run serve
```

### API文档

https://github.com/Binaryify/NeteaseCloudMusicApi

## 功能
1. 登录
2. 退出登录
3. 用户信息, 歌单，收藏，数量   
4. 用户歌单   
5. 用户等级  
6. 用户头像  
7. mv播放  
8. mv详情   
9. mv评论  
10. mv收藏  
11. mv点赞  
12. mv进度条  
13. 主页推荐歌单  
14. 歌单详情  
15. 歌曲播放  
16. 歌单播放顺序   
17. 歌单随机播放  
18. 歌单顺序播放  
19. 单曲循环播放  
20. 删除歌单
21. 歌曲歌词  
22. 歌曲进度条  
23. 选择上一首歌曲  
24. 选择下一首歌曲  
25. 搜索功能  
26. 综合搜索  
27. 单曲搜索
